# Как запустить бота — пошагово

---

## Шаг 1 — Создай бота в Telegram

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя (например: `Мои траты`) и юзернейм (например: `mytrackerbot`)
4. BotFather пришлёт токен — сохрани его, он нужен на следующих шагах

---

## Шаг 2 — Узнай свой Telegram ID

1. Найди в Telegram бота **@userinfobot**
2. Напиши ему `/start`
3. Он пришлёт твой ID — это число, например `123456789` — запиши

---

## Шаг 3 — Получи ключ Anthropic API (для фото чеков)

1. Зайди на https://console.anthropic.com/
2. Зарегистрируйся (Google-аккаунт подойдёт)
3. Слева — **API Keys** → **Create Key**
4. Скопируй ключ, начинается на `sk-ant-...`

> Без этого ключа фото чеков работать не будет. Стоит копейки.

---

## Шаг 4 — Залей код на GitHub

1. Зайди на https://github.com и создай аккаунт (бесплатно)
2. Нажми **New repository** → назови `expense-bot` → **Create repository**
3. Скачай [GitHub Desktop](https://desktop.github.com/) и установи
4. В GitHub Desktop: **Add** → **Add Existing Repository** → выбери папку `Трата расходов`
5. Нажми **Publish repository** → выбери репозиторий `expense-bot` → **Publish**

---

## Шаг 5 — Задеплой на Railway

1. Зайди на https://railway.app → **Login with GitHub**
2. Нажми **New Project** → **Deploy from GitHub Repo** → выбери `expense-bot`
3. Railway создаст проект. Зайди в него → вкладка **Variables**
4. Добавь переменные (кнопка **+ Add Variable**):

| Имя переменной | Значение |
|---|---|
| `BOT_TOKEN` | токен от BotFather |
| `ALLOWED_USER_ID` | твой Telegram ID |
| `ANTHROPIC_API_KEY` | ключ от Anthropic |

5. Перейди во вкладку **Volumes** → **New Volume** → примонтируй к пути `/data`
   > Это нужно чтобы данные не терялись при перезапуске

6. Нажми **Deploy** — Railway запустит бота

---

## Шаг 6 — Проверь

Открой Telegram, найди своего бота, напиши `/start` — он должен ответить.

---

## Как пользоваться

| Что пишешь | Что происходит |
|---|---|
| `50 кофе` | Записывает трату 50€ |
| `15.50 обед` | Трата с копейками |
| `+2000 зарплата` | Доход 2000€ |
| Фото чека | Бот распознаёт сумму и спрашивает подтверждение |
| `/balance` | Текущий баланс |
| `/week` | Итог этой недели |
| `/stats` | Итог месяца |
| `/history` | Последние 10 записей |
| `/del 5` | Удалить запись №5 |

Каждую **пятницу в 18:00** и **понедельник в 9:00** бот сам пришлёт отчёт по неделе.
